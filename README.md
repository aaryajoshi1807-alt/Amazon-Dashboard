1.Project Overview
The **Amazon Sales Dashboard** is an interactive Power BI project designed to analyze sales performance across regions, categories, customer segments, and time periods.  
The dashboard provides a clear overview of business performance and helps identify trends, patterns, and insights from Amazon sales data.

2.Project Objective
The objective of this dashboard is to:
- Monitor overall sales and profit performance
- Analyze regional sales distribution
- Identify top-performing product categories
- Understand customer segments and payment behavior
- Support data-driven business decisions

4.Data Description
The dataset contains Amazon sales information including:
- Order ID and Product ID
- Sales and Profit values
- Product Categories and Sub-Categories
- Customer Segments
- Payment Modes
- Regional and Country-level data
- Order Date and Ship Mode
